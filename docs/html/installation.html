InstallationÔÉÅ

This project was developed using the Poetry package manager and Python 3.12. To install the Python package and its dependencies, the following bash scripts can be used:

pip install poetry

OR

sudo apt-get install python3-poetry

The second form is necessary if the pip option returns an ‚ÄúThis environment is externally managed‚Äù error.

git clone https://github.com/yinchi/histopath-bim-des
cd histopath-bim-des

# if you have obtained a copy of histo.ifc
mkdir -p assets/private/
cp /path/to/histo.ifc assets/private/

# required by erdantic
sudo apt install libgraphviz-dev

# If 3.12 is not the default Python version
poetry venv use /usr/bin/python3/12

poetry install
